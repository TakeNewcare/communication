아두이노, usb선 

usb 포트를 통해서 내부적으로 포트가 할당되고 이를 통해 통신을 한다.

21강 아두이노에서 pc로 데이터 전달 
25강 디스플레dl(TM1637 - FND 모듈)
28강 온습도(DHT-11)

=> 디스플레이로 출력하는 것을 아두이노 끼리 할거냐 pc:N아두이노로 할거냐
45강 - pc1:아두이노N 통신

=> 데이터 엑셀 파일로 저장, 데이터베이스로 저장



바이트 통신
1바이트 => 0~255 자리까지 표현이 가능하다 => 4자리를 표현하려면 2바이트를 사용해야함
2바이트 => 0~65000

바이너리 자릿수는 내가 표현하고자 하는 숫자 N
몇 비트가 필요하냐 = log2N

EX => 표현 숫자 9999
